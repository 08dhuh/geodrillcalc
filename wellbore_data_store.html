Wellbore Data Store¶

This section covers the management and storage of wellbore data.

class geodrillcalc.wellbore_data_store.WellBoreDataStore(is_production_well: bool, logger=None)¶

Bases: object

WellBoreDataStore is a centralised aggregate class designed to store and manage input/output parameters related to wellbore installation and completion. It provides a structured way to initialise input parameters and store calculation results generated by external pipeline modules, such as specification and cost calculation modules. This class aims to provide attributes that ensure all relevant data is readily accessible for downstream processes.

Names of data parameters required for the wellbore calculations.
